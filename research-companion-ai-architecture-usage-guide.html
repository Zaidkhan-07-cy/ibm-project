🚀 Quick Start

Option A — Launcher Script (recommended)

# Navigate to the project folder
cd research_companion
python launch.py

Option B — Direct Streamlit

cd research_companion
pip install -r requirements.txt
streamlit run app.py

Option C — Windows Double-Click

Double-click: research_companion\run.bat

Then open http://localhost:8501 in your browser.

🏗️ Architecture

UI Layer — Streamlit app (app.py)
6 tabs: Literature Review · Papers · Trends · Visualizations · AI Chat · Multimodal Input
Agent Layer — agents/research_agent.py
Orchestrates pipeline: search → synthesise → detect trends → predict → Q&A
AI Core — core/groq_client.py
Groq LLaMA 3.3 70B: literature synthesis · trend detection · streaming Q&A · insight extraction
Data Layer — core/research_aggregator.py
arXiv Atom API · Semantic Scholar Graph API · CrossRef REST API → deduplicate → rank
Multimodal — core/multimodal_processor.py
PDF (pdfplumber) · DOCX (python-docx) · Images · Paste text · Chart descriptions
Visualisation Engine — utils/visualizations.py
9 interactive Plotly charts: cluster maps · citation networks · radars · heatmaps · gauges

✨ Key Capabilities

1

Academic Source Fusion

Aggregates papers from arXiv, Semantic Scholar (200M+ papers), and CrossRef in parallel. Deduplicates and ranks by citation count + recency. Falls back to demo data if APIs are unavailable.

2

Multimodal Research Input

Accepts PDF papers, DOCX reports, uploaded images, pasted text, research notes, and chart descriptions. Extracts key claims, methodology, gaps, and keywords using LLaMA 3.3.

3

Predictive & Agentic Layer

Detects 5–7 emerging trends, identifies declining topics, finds citation gaps, generates future research directions, and produces a field-sentiment score with confidence rating.

4

Interactive Visual Dashboard

9 interactive Plotly charts: topic cluster scatter map, citation network graph, publication timeline, trend radar, keyword frequency, sentiment gauge, citation gap heatmap, source distribution pie, and future directions sunburst.

5

Streaming AI Research Chat

Real-time Q&A grounded in loaded papers. Cites sources using [number] notation. Maintains multi-turn conversation history. Powered by Groq's ultra-low latency inference.
